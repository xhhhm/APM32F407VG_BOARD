## 开发板介绍

- 硬件信息：[开发板原理图/PCB链接](https://oshwhub.com/2239778894xhh/apm32f407)


该开发板常用 **板载资源** 如下：

- MCU：APM32F407IGT6，主频 168MHz，1MKB FLASH ，192+4KB RAM
- 外部 FLASH：有
- GD32F350G8 DAPlink
- 常用外设
  - LED：2个，（黄色，PE5/PE6）
  - 按键：2个，K1（PA1），K2（PA0）
- 常用接口：RS232转串口、USB SLAVE
